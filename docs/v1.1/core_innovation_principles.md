# 核心创新原理（v1.1）

## 1. 问题定义
遥感语义分割在真实部署中同时面临三类困难：
- 多模态异构：RGB 与 DSM 等模态统计差异大。
- 域偏移显著：训练域与目标域纹理、风格、类别比例不同。
- 结构细节脆弱：道路与建筑边界在跨域时最容易退化。

已有方法常分别优化其中一项，但缺少统一框架同时兼顾三者。

本版本主线聚焦 ISPRS 的同域与弱跨域场景，以保证结果稳定、可复验，再扩展到更强域偏移数据。

## 2. 核心思想
提出统一的协同优化框架，将三类能力在同一训练图中耦合：
- 模态协同：使用 MFNet 的双分支融合与参数高效 SAM 微调保持高表达能力。
- 域协同：在多层语义特征上施加对抗域对齐，压缩 source-target 分布差异。
- 结构协同：在伪标签高置信区域施加边界与对象一致性损失，抑制跨域边缘破碎。
- 提示协同：引入自动提示生成（auto-prompt）机制，用高置信区域生成点/框提示，提升 SAM 分支的伪标签质量与稳定性。

本质上是把语义一致性、域不变性、几何结构性做联合约束。

实验组织采用三段式课程：
1. same-domain 先验证结构约束有效性。
2. weak-cross-domain 再验证域对齐增益。
3. optional stress 最后验证强偏移鲁棒性。

## 3. 数学形式
设 source 样本为 (x_s, y_s)，target 样本为 x_t。
模型输出分割 logits 为 p，分层特征为 {f_l}。

总损失定义：

$$
\mathcal{L}_{total} = \mathcal{L}_{seg}^{src} + \lambda_{adv}\sum_{l \in \mathcal{S}}\mathcal{L}_{adv}^{(l)} + \lambda_{bdy}\mathcal{L}_{bdy}^{tgt} + \lambda_{obj}\mathcal{L}_{obj}^{tgt} + \lambda_{prm}\mathcal{L}_{prompt}^{tgt}
$$

其中：
- $\mathcal{L}_{seg}^{src}$：source 监督分割损失（CE 或 CE+Dice）。
- $\mathcal{L}_{adv}^{(l)}$：第 $l$ 层特征的域对抗损失（通过 GRL 与域判别器实现）。
- $\mathcal{L}_{bdy}^{tgt}$：target 伪标签边界损失（仅高置信区域）。
- $\mathcal{L}_{obj}^{tgt}$：target 对象一致性损失（仅高置信区域）。
- $\mathcal{L}_{prompt}^{tgt}$：自动提示生成后的一致性约束损失，用于降低伪标签噪声传播。

建议首版默认设置：
- $\mathcal{S}=\{high\}$（先单层稳定跑通），再扩展到 $\mathcal{S}=\{mid, high\}$。
- $\lambda_{prm}=0.05$（起始值，和 $\tau$ 联动调参）。

## 4. 为什么有效

### 4.1 域对齐解决全局偏移与局部偏移
语义高层特征对纹理扰动不敏感，对其施加对抗对齐可降低跨域统计差；加入中层对齐可补足小目标与细边界。

### 4.2 结构损失补足对抗训练短板
仅做域对齐通常会牺牲细节边界。边界与对象损失提供局部几何先验，可恢复道路边缘和建筑轮廓。

### 4.3 多模态融合提供稳定锚点
DSM 等几何信息在跨域时更稳定，作为形状约束来源可提升泛化上界。

### 4.4 参数高效微调提升可迁移性
LoRA/Adapter 仅更新少量参数，减少过拟合风险，在小样本或跨域场景中更稳。

### 4.6 自动提示降低伪标签噪声
基于高置信区域自动生成提示，可将伪标签从“纯阈值筛选”提升为“阈值+结构先验”联合筛选，在弱监督与跨域场景更稳。

### 4.5 课程式验证提升结论可信度
先在可控难度数据上完成机制验证，再逐步提升难度，可显著降低“高难数据低绝对分”对结论解释性的干扰。

## 5. 与现有项目关系
- MFNet 提供强多模态编码与 SAM 参数高效微调底座。
- GLGAN 提供可复用的域对抗优化思路。
- SAM_RS 提供可复用的边界与对象结构约束。
- RSPrompter 与 Auto-prompting SAM 提供可借鉴的提示学习与自动提示生成思路。
- 多级/相邻尺度多模态融合工作提供可借鉴的跨层融合与对齐设计。

创新点不在单模块发明，而在统一训练目标下的协同耦合与系统验证。

## 6. 预期实验现象
- target mIoU 提升：相比纯监督 MFNet 有稳定增益。
- 边界 F1 提升：道路与建筑类别边缘更连续。
- 参数效率更优：LoRA/Adapter 以更少可训练参数达到接近或更优精度。
- 泛化方差降低：不同随机种子下波动减小。
- 自动提示有效时：伪标签有效像素占比提升、早期训练振荡下降。

主线验收应优先看 weak-cross-domain 的相对提升，而非强域偏移数据上的绝对 mIoU。

## 7. 可写成论文贡献的表达模板
1. 提出一个面向遥感同域到弱跨域迁移的多模态统一框架，实现多层域对齐与结构先验的协同优化。
2. 提出针对 target 高置信区域的自动提示伪标签机制，将提示学习引入跨域语义分割训练，缓解伪标签噪声与边界退化。
3. 通过课程式实验设计验证精度、边界质量、参数效率与稳定性优势，并给出完整消融分析。

## 8. v1.1 实验实施证据规范
为保证复验与论文可追溯性，实验过程中必须固定保存以下证据。

### 8.1 必存数据（每次训练）
- 运行配置快照：完整 yaml、命令行参数、git commit id、随机种子、环境信息。
- 训练曲线原始数据：每 iter 与每 epoch 的 loss/metric（csv 或 parquet）。
- 伪标签统计：高置信像素占比、各类伪标签比例、阈值命中率。
- 域判别器统计：域分类准确率、对抗损失曲线、各层判别器损失（若启用多层）。
- 最优与最后模型：best.pth、last.pth、optimizer/scheduler 状态。
- 验证与测试结果：source/target 全量指标 json 与每类 IoU 明细。

### 8.2 必做图片
- 主曲线图：train/val 的 L_seg、L_adv、L_bdy、L_obj、L_prompt。
- 指标趋势图：source mIoU、target mIoU、Boundary F1 随 epoch 变化。
- 伪标签质量图：不同 epoch 的伪标签可视化与置信度热力图。
- 结构质量图：道路与建筑边界局部放大图（至少 4 组对照）。
- 失败案例图：每个协议至少 3 个 failure case，并附简短原因标签。

### 8.3 必做表格
- 主结果表：same-domain、weak-cross-domain、optional stress 三协议主指标。
- 消融表：-adv、-bdy、-obj、-prompt、single-level adv、multi-level adv。
- 参数效率表：Full fine-tune、LoRA、Adapter 的参数量、显存、耗时、精度。
- 稳定性表：3 seeds 的均值、标准差、最优值、最差值。

## 9. v1.1 局限与后续方向
- 局限
  - 伪标签质量依赖阈值，早期训练不稳定时噪声较大。
  - 当前域对齐主要在全局特征层，类别级对齐不足。
  - 在强域偏移数据上绝对精度可能仍不高，需要分开解读相对增益与绝对性能。
- 后续
  - 类别感知域对齐。
  - 不确定性感知伪标签重加权。
  - 频域一致性与空间结构一致性联合优化。
  - 将 LoveDA 等高难数据作为补充压力测试，而非主线结论来源。
